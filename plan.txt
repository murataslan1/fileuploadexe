# Claude Code Plan: Windows Forms — Fotograf Birlestirici & PDF Olusturucu

## Proje Ozeti

Kullanicilarin fotograflari (JPEG/PNG/BMP/HEIC) surukle-birak ile yukleyip, siralayip, tek bir PDF dosyasina birlestirip onizleme yaparak disa aktarabilidigi bir **Windows Forms (.NET 8)** masaustu uygulamasi.

## Teknik Mimari

### Teknoloji Secimi
- **Framework:** .NET 8 Windows Forms (C#)
- **PDF Kutuphanesi:** `PdfSharpCore` (MIT lisans, cross-platform PDF olusturma)
- **Goruntu Isleme:** `System.Drawing.Common` + `ImageSharp` (HEIC destegi icin)

public class PdfSettings

### ASAMA 3: PDF Olusturma Motoru (Oncelik: Yuksek)

**Gorev 3.1 -- PdfService.cs**
```csharp
public class PdfService
{
    // Ana birlestirme metodu
    public byte[] MergeImagesToPdf(
        List<ImageItem> images,
        PdfSettings settings);

    // Tek bir resmi PDF sayfasina ekle
    private void AddImageToPage(
        PdfDocument doc,
        string imagePath,
        PdfSettings settings);

    // Resmi sayfa boyutuna gore olcekle
    private (double x, double y, double w, double h) CalculateImagePlacement(
        double imgWidth, double imgHeight,
        double pageWidth, double pageHeight,
        double margin, ScaleMode mode);

    // PDF'i dosyaya kaydet
    public void SavePdf(byte[] pdfBytes, string outputPath);

    // PDF'i gecici dosyaya kaydet (onizleme icin)
    public string SaveToTempFile(byte[] pdfBytes);
}
```

**Gorev 3.2 -- Olcekleme Algoritmasi**
```
FitToPage:
  1. Resim en-boy oranini koru
  2. Marjinleri cikar -> kullanilabilir alan hesapla
  3. Genislik orani = kullanilabilir_genislik / resim_genislik
  4. Yukseklik orani = kullanilabilir_yukseklik / resim_yukseklik
  5. min(genislik_orani, yukseklik_orani) kullan -> merkezle

FitToWidth:
  1. Genislige sigdir, yukseklik orantili
  2. Sayfa yuksekligini asarsa -> otomatik kirp veya yeni sayfa

OriginalSize:
  1. Resmi orijinal DPI'da yerlestir
  2. Sayfadan tasarsa -> merkezle ve kirp
```

**Gorev 4.2 -- Onizleme Yaklasimi (2 Secenek)**

**Secenek A -- PictureBox ile sayfa gorseli (Basit, Onerilen):**
```
- PDF olustur -> her sayfayi Bitmap'e render et
- PictureBox.Image = renderEdilmisSayfa
- Navigasyon: < Onceki | Sayfa X/Y | Sonraki >
- Avantaj: Ek bagimlilik yok
- Dezavantaj: Gercek PDF deneyimi degil
```

**Secenek B -- WebView2 ile PDF Viewer (Zengin, Opsiyonel):**
```
- PDF'i temp dosyaya kaydet
- WebView2.Source = new Uri(tempPdfPath)
- Chrome tabanli tam PDF goruntuyeci
- Avantaj: Zoom, arama, yazdirma destegi
- Dezavantaj: WebView2 Runtime bagimliligi (~150MB)
```

### ASAMA 5: Is Akisi ve Event Yonetimi (Oncelik: Orta)

**Gorev 5.1 -- Ana Akis**
```
1. Kullanici fotograflari surukle-birak veya dosya sec ile yukler
2. Her fotograf icin:
   a. Dosya turu kontrolu (desteklenen format mi?)
   b. HEIC ise -> JPEG'e donustur
   c. EXIF orientasyon duzelt
   d. Thumbnail olustur
   e. ListView'e ekle
3. Kullanici siralamayi ayarlar (drag-reorder / yukari-asagi)
4. Kullanici sayfa ayarlarini secer (A4, yonelim, olcekleme)
5. "Birlestir & Onizle" butonuna tiklar
6. Arka planda PDF olusturulur (BackgroundWorker / Task.Run)
7. ProgressBar ilerler
8. PDF onizlemede gosterilir
9. "PDF Kaydet" -> SaveFileDialog -> dosya kaydedilir
```

**Gorev 5.2 -- Hata Yonetimi**
```
- Bozuk dosya: Hata mesaji goster, digerlerine devam et
- Cok buyuk dosya (>50MB): Uyari ver, sikistirma oner
- Bellek yonetimi: Buyuk resimleri stream ile isle, dispose et
- Bos liste: "Birlestir" butonunu devre disi birak
- PDF olusturma hatasi: try-catch + kullaniciya anlamli mesaj
```

## Dikkat Edilmesi Gerekenler

1. **Bellek Yonetimi:** Buyuk resimleri `using` blogu icinde isle, Bitmap'leri mutlaka `Dispose()` et
2. **Thread Safety:** UI guncellemelerini `Invoke`/`BeginInvoke` ile yap
3. **Gecici Dosyalar:** Temp klasorunde olusturulan dosyalari uygulama kapanirken temizle
4. **HEIC Destegi:** Windows'ta varsayilan olarak desteklenmez, ImageSharp ile donustur
5. **DPI Farkindaligi:** `Application.SetHighDpiMode(HighDpiMode.SystemAware)` ekle
6. **PDF Boyutu:** Resimleri PDF'e eklerken JPEG sikistirma uygula (kalite: 85-90)
